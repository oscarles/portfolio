Covo'IUT — Covoiturage

Plateforme web PHP / MySQL Projet de groupe

Présentation

Plateforme de covoiturage dédiée aux étudiants de l'IUT d'Amiens, développée en équipe dans le cadre d'une SAE. L'application permet de proposer et réserver des trajets, de gérer son profil, et inclut un espace d'administration complet pour les modérateurs.

Fonctionnalités

  • Création et publication de trajets avec date, heure et places disponibles
  • Système de réservation avec confirmation et annulation
  • Recherche de trajets avec autocomplétion d'adresse (API externe)
  • Gestion de profil utilisateur avec photo et historique
  • Système d'avis et notations entre conducteurs et passagers
  • Espace admin : gestion des utilisateurs, modération des trajets et signalements
  • Messagerie interne entre utilisateurs

Points techniques

  • Architecture PHP avec routeur custom et gestion des sessions
  • Base de données MySQL avec relations et contraintes d'intégrité
  • Design responsive avec Bootstrap 5
  • Requêtes AJAX pour la mise à jour dynamique sans rechargement

Stack technique

PHP MySQL Bootstrap 5 JavaScript AJAX API REST
